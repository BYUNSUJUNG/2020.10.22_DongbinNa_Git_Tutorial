## 1. \# 이용하여 글씨 크기 조절
**\#은 6개까지 사용가능**

# \# 1개 사용
## \# 2개 사용
### \# 3개 사용
#### \# 4개 사용
##### \# 5개 사용
###### \# 6개 사용
####### \# 7개 사용인데, 적용안됨

## 2. 소스코드 블록 사용
**자판 1옆에 `를 입력하여 생성**

```c
  #include <stdio.h>
  
  int main(void) {
    print("Hello World!");
    return 0;
  }
```
